# Legacy Arşivi

Bu klasör, aktif kod tabanını sade tutmak için taşınan eski dosyaları içerir.

## Klasörler
- `legacy/app/` → eski monolitik uygulama dosyaları (`leta_app.py` vb.)
- `legacy/installer/` → eski installer sürümleri
- `legacy/backups/` → geçmiş yedek klasörleri
- `legacy/docs/` → geçmiş notlar, iç dökümanlar ve taşıma planları

## Not
Aktif uygulama geliştirmesi `script/` klasörü ve kökteki `.spec` dosyaları üzerinden devam eder.
